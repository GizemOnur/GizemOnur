FestiWorld Project

Creation of the front-end for a realistic WebSite as a personal project using React js & Material UI as part of the BeCode Web Developer training. FestiWorld is a Website for electronic music & festival lovers so that they can find the ideal festival for them in Europe and create a community around their passion. Realization of the front-end in 4 weeks with a session system: registration, connection / disconnection, search bar, sort by filters etc, events posts, news etc.
